WORLD GYM

WorldGym es un proyecto que utiliza APIs públicas para ofrecer una experiencia dinámica donde puedes crear y gestionar rutinas personalizadas para principiantes e intermedios, guardar ejercicios favoritos y adaptarlos según tus necesidades.

Visitar sitio

Objetivos del proyecto

1.

Crear una plataforma interactiva que permita a los usuarios crear y gestionar rutinas personalizadas para diferentes niveles de entrenamiento.

2.

Integrar APIs públicas para ofrecer información actualizada y variada sobre ejercicios y rutinas de gimnasio.

3.

Facilitar la gestión de ejercicios favoritos y rutinas guardadas, mejorando la experiencia y motivación del usuario.

Sobre la página

  • ¿Cuál fue el proceso de elaboración y desarrollo del proyecto?

    El proyecto consistió en utilizar APIs públicas para crear una plataforma dinámica que permite a los usuarios crear y gestionar rutinas personalizadas según su nivel de entrenamiento.

  • ¿Cómo se diseñó la experiencia de usuario y la interfaz?

    Se diseñó una interfaz intuitiva y funcional, que facilita la navegación y gestión de rutinas, adaptada para ofrecer una experiencia atractiva en todos los dispositivos.

  • ¿Qué impacto o beneficios se esperan para los usuarios o la comunidad?

    Se espera que la plataforma motive a los usuarios a mantenerse activos, facilitando la creación y seguimiento de rutinas personalizadas y mejorando su experiencia de entrenamiento.
